# Microservice-Email
    Microservice for sending email

  ## Built with
  - 🍃 SpringBoot
  - 🍃 Java 11
  - 🍃 SpringMail
  - 🍃 Feing
  - 🌶 Lombok
  - 🍃 Postgres
  
  ## Starting
  - Clone or download the project
  - Import the project into your preferred IDE like IntelliJ for example.
  - start the project
  - Use a platform like postman to test routes
